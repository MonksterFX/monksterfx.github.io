projects#

In the past years I worked on multiple different side projects. Some you find on the projects page. I really love to prototyp or to try different technologies. I like build micro apps link my encrypted note service throwAway.js and my seasonal calendar for fruits and vegatables.

But beside building microtools here is a short introduction into my biggest ongoing three side projects:

klingl.me

klingl.me is a connected paper doorbell. Jup this is a pretty uncommon description. So what did we do: We are building a open platform for DIY IoT doorbells. Most people do not know how to build a IoT Device thats why we provide a basic App (React Nativve). You just have to provide a QR Code with your klingl-url and stick it to any place you want. Know you have a paper door bell, with muting, sharing the klingl with other people, MQTT or Webhooks… If you want to learn more visit klingl.me.

wg-couch.de

Searching new flatmates is quite annoying. So in 2019 me and a group of friends did a hackathon to try to solve this communication overhead. It was a fun journey, but we could not bring this further than an prototyp. Now 3 Years later a friend of mine and I want to revive that idea. So we currently are in a complete rewrite of this hackathon project.

If you want to visit the old page and application go to wg-couch.de. The chances something is broken are big but rewrite is on the way.

coffee-scale-x

Coffe nerd? There are a lot of fancy coffee scales out there but for me the missed a detail. How does the flow of water behave in the filter. Quite a nerdy question but yep thats why this project exists.

Coffee-Scale-X is an open source project which contains all the stuff you need to build a fancy coffee scale with less than 100 bucks. You will find all the CAD files, software and the source files for the Coffee-Scale-X-App to use this scale in github.

berlin meetups tips#

React Meetup Berlin

It’s amazing to see how many meetups in berlin have such a good quality. For React people I can really recommend the React Berlin - Meetup. Good speakers and talks about react.js and react native.

TypeScript and JavaScript

Prisma is a great company which doing an awesome job with there library. The are also organizers of different meetups. TypeScript - Meetup is definitiv worth a visit!

insights from startups

The startup grammarly host some pretty interesting tech talks with there own format called Grammarly Tech Talks.

visual art

If you are into “creative code” and want see some stunning computer generated visuals, try to visit on of the events provided by Creative Code Berlin - Meetup or check out there pretty cool site Creative Code Berlin. Well structured meetup with all information saved to an pad where you can revisit all the cool stuff you saw.
